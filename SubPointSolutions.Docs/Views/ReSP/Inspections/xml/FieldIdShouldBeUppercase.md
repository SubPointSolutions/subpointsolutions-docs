---
Title: Incorrect 'ID' attr name
FileName: resp515105.html
---
### Description
List scoped field MUST HAVE "ID" (not "Id") attribute. SharePoint generates random GUID for these fields.

### Resolution
Please update ID attribute. Make it upper-case.
